Welcome To R.E.A.S.N an acronym for Redundancy Elimination Algorithm for Sensor Network

Based on the concept of Entropy which is the total amount of average information fetched from the data transmission

This project could prove to be very vital in sensor prioritization based on the present and the previous data fetched from the sensor.

This project at present is in the development phase. Developed by three Undergraduate Students.
